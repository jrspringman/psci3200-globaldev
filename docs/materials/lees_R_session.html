Live R Coding Session

Author

Jeremy Springman

Published

December 3, 2024

Overview

This LEES Live Session will focus on using R and RStudio for data analysis. During this session, we will cover the following topics:

Activity Description Duration
Introductions and Sign-in Fill out this form to sign into the session. 5 min
R Coding Review Review of basic R concepts covered in Module 2 Week 2.2 10 min
Cleaning Data Read-in real data, create new variables, calculate quantities 30 min
Data Visualization Create tables and figures 10 min
Regressions Run regressions and store output in a presentable format 10 min
Q&A Ask questions and writing code or conducting analysis 25 min

Please consult this document for a more detailed overview of what will be covered.

Required Preparation

Before attending the live session, participants should have R and RStudio installed and have RStudio opened on their personal computer. Module 2 Week 2.2 included instructions on installing R and RStudio and provided a basic introduction to these tools.

Materials

Before attending the live session, please download the R Tutorial Source script using the link below. I will be using the R Tutorial Source file (.qmd) to lead the session. The R Tutorial Visual contains a link to a rendered version of the R Tutorial Source that you can open as an .html document in your web browser. If you prefer, you can copy and paste code from the html file into RStudio.

Optional Preparation

I will be using a Quarto file (.qmd) to lead the session. Quarto is a specific type of file in RStudio that integrates code into document production. Quarto can be useful for producing .pdf or .html files that to share the results of data analysis with donors and policymakers.
